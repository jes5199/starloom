============
Installation
============


Stable release
--------------

To install StarLoom 2025, run this command in your terminal:

.. code-block:: console

    $ pip install starloom_2025

This is the preferred method to install StarLoom 2025, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for StarLoom 2025 can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/jes5199/starloom_2025

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/jes5199/starloom_2025/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/jes5199/starloom_2025
.. _tarball: https://github.com/jes5199/starloom_2025/tarball/master
